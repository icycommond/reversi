# React 黑白棋游戏

[在线体验](https://icycommond.github.io/reversi/)

## 项目简介
这是一个使用现代 Web 技术栈开发的黑白棋（Reversi/Othello）游戏。游戏支持双人对战，具有简洁直观的界面和流畅的游戏体验。

## 技术栈
- TypeScript
- React
- Tailwind CSS
- Vite
- Node.js

## 特点
- 经典的黑白棋游戏规则实现
- 响应式设计，支持多种设备
- 简洁优雅的用户界面
- 实时显示合法落子位置
- 本项目代码主要由 Cursor AI 辅助完成

## 如何运行

1. 克隆项目
git
git clone https://github.com/icycommond/reversi.git

2. 安装依赖
npm install

3. 运行项目
npm run dev
